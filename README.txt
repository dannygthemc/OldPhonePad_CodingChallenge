The provided program has two main methods.

One allows the user to enter their own string to be processed.

The other runs the provided test suite.

By Default, User Input Mode is active.

To Switch between the modes, Access: OldPhoneApp.csproj and comment out the currently uncommented StartupObject line
And Uncomment out the other

You can use the CTRL + / command with the line selected to quickly faciliate this.

The current dotNet version is set to 8.0. If Running a different version locally, can adjust this value in OldPhoneApp.csproj under the TargetFramework value
